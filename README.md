# Portfolio Website

A clean, responsive portfolio site that showcases design and development work.

## Overview

- Pages: Home, About, Dev, Design
- Tech: HTML, CSS (Bootstrap + custom styles), JavaScript
- Highlights: retro/pixel typography, dark/light themes, responsive layout, and a simple project carousel

## Structure

```
.
├── index.html       # Home
├── about.html       # About
├── dev.html         # Development projects & tech stack
├── design.html      # Design work / case studies
├── style.css        # Custom styles and theme variables
├── script.js        # UI interactions (e.g., carousel)
└── src/             # Images and other static assets
    ├── profile-img.png
    ├── poster-development.jpg
    ├── poster-design.jpg
    └── project-1.jpg
```

## Development

This is a static site. Open `index.html` in a modern browser or serve the folder with any static file server.
No build tools are required.

## Deployment

Built to be hosted on any static hosting provider (e.g., GitHub Pages, Netlify, Vercel). Simply deploy the
root folder; all assets are relative.
